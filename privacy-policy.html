Privacy Policy — Streaml ConnectIN Social Media Assist

Last updated: February 25, 2025

Single purpose. This extension's purpose is to assist with user-initiated messaging and connection requests on LinkedIn and X (Twitter) using user-provided templates and content from the Streaml platform. It is unofficial and not affiliated with or endorsed by LinkedIn Corporation, X (Twitter), or Meta.

Data Collection

We do not collect, transmit, or sell any personal data.

The extension stores only minimal configuration in the browser's local storage, for example:

  • Extension mode preference (open or closed).
  • Activity logs for user review and export.
  • Queue status and processing state.

These values remain on the user's device in chrome.storage.local and are never transmitted to our servers or any third parties.

Important: All messaging content, profile URLs, and personal information are processed locally on your device. We have no access to your messages, contacts, or any data you interact with through this extension.

Permissions & Purpose

  • scripting — inject scripts into LinkedIn, Instagram, and X (Twitter) pages to assist with filling message forms and connection requests. Scripts only run on the specific social media platforms.
  • storage — save local preferences and activity logs on your device only.
  • downloads — enable users to export their own activity logs on demand.
  • host_permissions — limited to:
    • https://www.linkedin.com/* — for LinkedIn messaging and connection features
    • *://x.com/* — for X (Twitter) direct messaging
    • *://twitter.com/* — for Twitter legacy URL support
    • *://instagram.com/* — for Instagram messaging features
    • https://app.streaml.app/* — for integration with Streaml platform
    The extension can only operate on these specific domains.

Each permission directly supports the extension's functionality. No broader access is requested, and all operations require explicit user confirmation.

External Services

No external services are used. This extension does not use analytics, advertising services, or any third-party data collection services. All functionality operates locally on your device.

The extension only communicates with:

  • LinkedIn — for messaging and connection features (user-initiated only)
  • X (Twitter) — for direct messaging (user-initiated only)
  • Instagram — for messaging features (user-initiated only)
  • Streaml Platform — for receiving profile URLs and messages (user-initiated only)

We do not proxy, log, or store any of these communications.

Security

  • All data and preferences remain on your device via chrome.storage.local.
  • Only your browser communicates directly with LinkedIn, X (Twitter), Instagram, and Streaml.
  • Scripts run only on declared host sites as enforced by Chrome Extensions.
  • All actions require explicit user confirmation before execution.
  • No data is transmitted to our servers or any third parties.

Data Retention & Deletion

  • Settings persist until the user clears extension data or uninstalls the extension.
  • Uninstalling the extension deletes all locally stored data.

Children’s Privacy

This extension is not directed to children under 13 and does not knowingly collect data from children.

Changes to This Policy

We may update this policy. Any updates will be posted here, with the “Last updated” date adjusted accordingly.
