Privacy Policy — ConnectIN Social Media Assist

Last updated: 2025-08-26

ConnectIN Social Media Assist is a Chrome extension that helps users automate sending and commenting on social media (LinkedIn, Twitter/X, Instagram) with AI-generated text. This extension is unofficial and not affiliated with or endorsed by LinkedIn Corporation, Twitter Inc. (X), or Instagram/Meta.

Data Collection

We do not collect, transmit, or sell any personal data.

The extension stores only minimal configuration in the browser’s local storage, such as:

  • popupMode — whether automation is “Open” or “Closed”.

Permissions & Purpose

  • tabs — open profile/post URLs in background tabs to perform outreach.
  • scripting — inject scripts that read profile/post context and fill in messages or comments.
  • storage — save user preferences (API key, persona, mode, logs toggle).
  • downloads — allow users to export logs of their own campaign activity.
  • activeTab — support one-click actions on the currently open tab.
  • host_permissions — restricted only to linkedin.com, x.com / twitter.com, instagram.com, and the AI API domain.

Each permission is essential for the extension’s single purpose of outreach automation. No broader access is requested.

External Services

The extension does not use analytics or advertising networks. If the user enables AI features, requests are sent directly from their browser to the AI provider using the API key they supplied. We do not proxy, log, or store these requests.

Security

  • Data remains on the user’s device via chrome.storage.local.
  • Only the user’s browser communicates with social media sites and the AI API.
  • We follow the Chrome Extensions security model: scripts run only on declared host sites.

Data Retention & Deletion

  • Settings persist until the user clears extension data or uninstalls the extension.
  • Uninstalling the extension deletes all locally stored data.

Children’s Privacy

This extension is not directed to children under 13 and does not knowingly collect data from children.

Changes to This Policy

We may update this policy. Any updates will be posted here, with the “Last updated” date adjusted accordingly.
