Privacy Policy — ConnectIN Social Media Assist

Last updated: 2025-08-29

Single purpose. This extension’s single purpose is to assist user-initiated messaging and connection requests on LinkedIn (and X) using user-provided templates. It is unofficial and not affiliated with or endorsed by LinkedIn Corporation or X (Twitter).

Data Collection

We do not collect, transmit, or sell personal data.

The extension stores only minimal configuration in the browser’s local storage, for example:

  • popupMode — whether automation is “Open” or “Closed”.
  • Saved message template/persona.
  • Optional AI API key (if the user enables AI messaging).
  • Optional log/export toggle.

These values remain on the user’s device in chrome.storage.local and are not transmitted to our servers.

Permissions & Purpose

  • tabs — open profile/post URLs in background tabs and detect when pages are ready for user confirmation.
  • scripting — inject a small script on LinkedIn/X to prompt for confirmation, fill the message, and click send.
  • storage — save the local preferences listed above.
  • downloads — let the user export their own activity log on demand.
  • host_permissions — limited to:
    • https://www.linkedin.com/*
    • https://x.com/* (and/or https://twitter.com/* if applicable)
    to read the page and place the message.
  • externally_connectable — limited to https://app.streaml.app/* so that only our site can pass user-provided URLs/messages to the extension.

Each permission directly supports the extension’s single purpose. No broader access is requested.

External Services

No analytics or advertising services are used. If the user enables AI features, requests are sent directly from their browser to the AI provider using the user’s own API key. We do not proxy, log, or store these requests.

Security

  • All preferences remain on the device via chrome.storage.local.
  • Only the user’s browser communicates with LinkedIn/X and (optionally) the AI API.
  • Scripts run only on declared host sites as enforced by Chrome Extensions.

Data Retention & Deletion

  • Settings persist until the user clears extension data or uninstalls the extension.
  • Uninstalling the extension deletes all locally stored data.

Children’s Privacy

This extension is not directed to children under 13 and does not knowingly collect data from children.

Changes to This Policy

We may update this policy. Any updates will be posted here, with the “Last updated” date adjusted accordingly.
